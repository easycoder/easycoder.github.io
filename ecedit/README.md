# EC Editor

The files here are for the EasyCoder editor, 'ecedit'. They comprise 5 files:
ecedit-server.ecs__
ecedit.html__
ecedit.ecs__
ecedit.json__
README.md (this file)__

Copy ecedit-server.ecs and ecedit.html to the root of your project.

Run the server from your project root:

`easycoder ecedit-server.ecs {port}`

where {port} is a port number such as 8080, then open the editor in your browser at:

`http://localhost:{port}/ecedit.html`

This should start the editor. You can browse and edit files across your entire project.


See the instructions in our [Primer](https://easycoder.github.io/primer.html).
